# Cat-Raising - 全方位貓咪生活管理平台

一站式貓咪生活管理應用程式，整合營養分析、健康記錄、智能提醒、支出管理等功能，基於 Next.js 14 和 Supabase 開發。

## 🚀 快速開始

### 1. 設置 Supabase 資料庫

#### 建立 Supabase 專案

1. 前往 [Supabase](https://supabase.com) 並登入/註冊
2. 點擊 "New Project"
3. 選擇組織並填寫專案資訊：
   - Name: `cat-nutrition-calculator`
   - Database Password: 設置安全密碼
   - Region: 選擇最近的區域
4. 等待專案建立完成（約 2-3 分鐘）

#### 執行資料庫 Schema

1. 在 Supabase Dashboard 點擊左側 "SQL Editor"
2. 複製 `supabase-schema.sql` 檔案的所有內容
3. 貼上到 SQL Editor 並點擊 "Run"
4. 確認所有表格建立成功

#### 獲取連線資訊

1. 點擊左側 "Project Settings" → "Data API"
2. 複製以下資訊：
   - `Project URL`
   - `anon public` key

### 2. 設置本地環境

#### 安裝依賴

```bash
npm install
```

#### 配置環境變數

編輯 `.env.local` 檔案：

```env
NEXT_PUBLIC_SUPABASE_URL=你的專案URL
NEXT_PUBLIC_SUPABASE_ANON_KEY=你的anon-key
```

#### 啟動開發伺服器

```bash
npm run dev
```

開啟 [http://localhost:3000](http://localhost:3000) 檢視應用程式。

## 📋 功能特色

### 🧮 營養管理
- **精準計算** - 基於 AAFCO 標準計算乾物質含量
- **多貓關聯** - 一個食品可同時關聯多隻貓咪，靈活管理餵食記錄
- **營養分析** - 自動評估蛋白質、脂肪、碳水化合物是否達標
- **食品比較** - 支援多款貓糧營養成分對比分析

### 🏥 健康記錄
- **醫療檔案** - 完整記錄疫苗接種、健康檢查、病歷資料
- **用藥管理** - 追蹤藥物使用時間、劑量與效果
- **體重監控** - 定期記錄體重變化，生成趨勢圖表
- **症狀日誌** - 記錄異常症狀與行為變化

### 📝 生活記錄
- **飲食日記** - 詳細記錄每日餵食時間、份量與食慾狀況
- **日常筆記** - 自由記錄貓咪行為、心情與特殊事件
- **照片記錄** - 上傳貓咪照片，建立成長回憶相簿

### ⏰ 智能提醒
- **餵食提醒** - 自訂餵食時間與頻率提醒
- **醫療提醒** - 疫苗、驅蟲、健檢等重要日期提醒
- **用藥提醒** - 準確的服藥時間與劑量提醒
- **清潔提醒** - 貓砂更換、環境清潔等日常任務

### 💰 財務管理
- **支出記錄** - 分類記錄飼料、醫療、用品等各項花費
- **預算管理** - 設定月度預算，追蹤支出狀況
- **成本分析** - 統計各類別支出比例與趨勢
- **發票管理** - 拍照儲存購買憑證

### 📦 庫存管理
- **消耗品追蹤** - 管理貓糧、貓砂、玩具等用品庫存
- **自動提醒** - 庫存不足時主動提醒補貨
- **採購清單** - 智能生成採購建議清單
- **使用統計** - 分析各類用品消耗速度

### 📊 數據分析
- **月度報表** - 自動生成健康、支出、餵食等各項報表
- **趨勢分析** - 長期數據趨勢分析與預測
- **個性化建議** - 基於數據提供客製化照護建議

### 🔧 系統功能
- **多貓管理** - 為每隻貓咪建立專屬檔案
- **雲端同步** - 資料自動備份，多裝置同步
- **PWA 支援** - 可安裝為手機 App，離線功能
- **個性頭像** - 16 款精美貓咪頭像選擇
- **響應式設計** - 完美適配手機、平板、桌面
- **🆓 完全免費** - 使用免費的雲端服務

## 🧮 計算公式

### 基礎計算

```
乾物質含量 = 100% - 水分含量%
```

### 乾物質基準營養成分

```
乾物質基準蛋白質 = (蛋白質% / 乾物質含量%) × 100%
乾物質基準脂肪 = (脂肪% / 乾物質含量%) × 100%
乾物質基準纖維 = (纖維% / 乾物質含量%) × 100%
```

## 🚀 部署到 Vercel

   ```
   NEXT_PUBLIC_SUPABASE_ANON_KEY = 你的Supabase anon key
   ```
4. 點擊 "Deploy"
